The open standards for electronic permitting and plan check.

Standards for streamlined permitting.

Open Permit defines standards for electronic interaction with the permitting and plan check processes of jurisdictions.

API

Standard way of retrieving permit requirements, applying for permits, submitting documents, retrieving plan check comments, scheduling inspections and a lot more.

PRDF

Permit Requirements Description Format is an Open Data format that can be used to specify permit requirements, including applicable fees, documents to be submitted, plan check disciplines and needed subpermits.

Open Source Libraries

Client libraries allow for easy manipulation of the Open Permit APIs and PRDF data.
